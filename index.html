Examples

  1. Basic - Basic loading vector tiles with debug enabled.
  2. Cache - Cache enabled.
  3. Click - Click to select one or multiple features.
  4. Custom draw - Custom draw function for each feature.
  5. Filter - Filter features by it properties.
  6. Hover - On mouse hover event.
  7. Layers - Add remove vector tiles layers.
  8. Point, linestring and polygon - Show all type of geometries.
  9. Preselected features - Set features as selected before loading.
  10. Style filter and layer - Update the style, filter and visible layers simultaneously.
  11. Styles feature - Style based on feature properties.
  12. Styles - Change style dynamically.
